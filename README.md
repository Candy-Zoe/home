# AI学习库汇总

这是一个用于学习各种人工智能库的项目，包含数据处理、机器学习、深度学习、计算机视觉、自然语言处理等多个领域的学习demo。

## 项目结构

### 1. 数据处理库

| 库 | 简介 | 用途 |
|----|------|------|
| **NumPy** | 数值计算基础库 | 数组操作、矩阵运算、傅里叶变换、随机数生成 |
| **Pandas** | 数据分析和处理库 | DataFrame操作、数据清洗、时间序列、数据聚合 |

### 2. 大数据处理库

| 库 | 简介 | 用途 |
|----|------|------|
| **PySpark** | Spark的Python API | 分布式计算、大规模数据处理、Spark SQL |
| **Dask** | 并行计算库 | 大数据处理、并行化Pandas/NumPy操作 |
| **Vaex** | 内存高效的大数据处理 | 处理TB级数据、延迟计算、可视化 |

### 3. 可视化库

| 库 | 简介 | 用途 |
|----|------|------|
| **Matplotlib** | 基础可视化库 | 绘制各种图表、数据可视化 |
| **Seaborn** | 统计数据可视化 | 高级图表、热力图、分布图 |
| **Plotly** | 交互式可视化库 | 交互式图表、仪表盘、3D可视化 |

### 4. 机器学习库

| 库 | 简介 | 用途 |
|----|------|------|
| **scikit-learn** | 经典机器学习库 | 分类、回归、聚类、降维、模型评估 |
| **XGBoost** | 高效梯度提升树 | 分类、回归、特征重要性、参数调优 |
| **LightGBM** | 微软开源梯度提升库 | 高效训练、大规模数据、并行计算 |
| **CatBoost** | Yandex开源梯度提升库 | 自动处理类别特征、避免手动编码 |

### 5. 深度学习库

| 库 | 简介 | 用途 |
|----|------|------|
| **PyTorch** | Facebook深度学习框架 | 张量操作、自动微分、神经网络、GPU加速 |
| **TensorFlow** | Google深度学习框架 | 张量操作、自动微分、神经网络、生产部署 |
| **PyTorch Lightning** | PyTorch训练框架 | 简化训练循环、分布式训练、日志记录 |

### 6. 强化学习库

| 库 | 简介 | 用途 |
|----|------|------|
| **Gymnasium** | OpenAI Gym的现代续作 | 经典控制任务、Atari游戏环境、自定义环境 |
| **Stable Baselines3** | 强化学习算法实现 | DQN、PPO、SAC等经典算法 |

### 7. 计算机视觉库

| 库 | 简介 | 用途 |
|----|------|------|
| **Pillow** | Python图像处理库 | 图像基本操作、滤镜、格式转换 |
| **OpenCV** | 最流行的计算机视觉库 | 图像读取、处理、特征检测、目标识别、视频处理 |
| **scikit-image** | 基于scikit-learn的图像处理库 | 图像分割、特征提取、形态学操作、滤波 |

### 8. 自然语言处理库

| 库 | 简介 | 用途 |
|----|------|------|

### 9. 图神经网络库

### 12. 其他工具库

| 库 | 简介 | 用途 |
|----|------|------|
